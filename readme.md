## Quotation System
This Quotation system is build using Laravel 5.2 PHP framework and AdminLTE template.
Tools: jQuery, DataTables js & Select2 js.

## Availabe Functions:
- Add Customers.
- Edit Customers.
- Add Products.
- Edit Products.
- Add Product stocks to inventory.
- Delete Product stocks.
